# Changelog

## v1.5.0 (2026-05-13)

### Новое
- **Локализация ru/en** — полный перевод интерфейса, мгновенное переключение без перезапуска
  - Все окна, кнопки, подписи полей, колонки таблиц
  - Читаемые названия ключей в Medicine Config и Liquid Config (без изменения JSON)
- **Автозагрузка GM_Config** — все конфиги открываются автоматически при запуске
  - Приложение само находит папку `GM_Config` внутри указанного пути проекта
  - Настройка пути через кнопку 📁 в боковой панели
- **Окно первого запуска** — выбор папки проекта при первой установке
- **Self-contained exe** — Python embeddable встроен в приложение, распаковывается автоматически

### Улучшения
- Кнопка языка EN/RU в боковой панели под названием программы
- Кнопка 📁 для смены пути к проекту без перезапуска
- Настройки (тема, язык, путь) сохраняются в `settings.json` рядом с exe

### Удалено
- Раздел **MAKE STASH** перенесён в отдельное приложение

## v1.2.0 (2026-05-13)

### Новое
- **Проверка ошибок** (`Ctrl+E`) — валидация всех разделов по справочнику предметов
  - Рецепты: RESULT, PLAN, инструменты, ингредиенты
  - Квесты: TYPE_OBJECT целей/наград, дублирующиеся ID
  - NPC: аттачменты, ID квестов, позиции
  - Пресеты лута, MineRock, Food, Rock Points
  - Экспорт отчёта в TXT
- **Undo/Redo** (`Ctrl+Z` / `Ctrl+Y`)
  - TextBox изменения
  - Снапшоты при удалении рецептов, квестов, NPC
- **Правый клик** на любом DataGrid/ListBox
  - 📋 Копировать ClassName
  - 🔍 Найти в справочнике
- **Карта DayZ** `[WIP]` — встроенная интерактивная карта
  - Маркеры NPC, триггеров и Rock Points
  - Показ координат X/Z при наведении
  - Поддержка нескольких карт через `map.json`

### Улучшения
- Кнопка `✓ Проверить` в статусбаре
- Статусбар: Статус · Несохранённые изменения · Ctrl+F · Проверить

### Исправления
- NullReferenceException при открытии ValidationWindow
- Бесконечный цикл при инициализации карты
